# Springbank Page-Type Framework

## Approved page types
1. Property home page
2. Open-home event page
3. Gallery page
4. Regional hub page
5. Town guide page
6. Buyer-guide article
7. Comparison article
8. Property spotlight article
9. Optional buyer-fit page
10. Optional trust / methodology page

## Enforced rules
- A page should have one primary intent.
- A new page must justify itself with a distinct user need.
- Town guides should not become generic blog posts.
- Property spotlight posts should be limited.
- FAQ-like answer blocks should usually live inside stronger pages, not as stand-alone thin pages.
